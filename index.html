Características

Solicitudes en Tiempo Real

Permite gestionar múltiples solicitudes en el chat en tiempo real, optimizando la interacción con los espectadores.

Personalización Completa

Configura el bot a tu gusto mediante archivos personalizados para adaptarse a tu estilo.

Integración Sencilla

Compatible con la API de osu! para obtener información precisa de cada mapa solicitado.

Documentación

Para usar ASC3NTbot, sigue estos comandos en el chat de Twitch:

  • !request nombre_del_mapa - Solicitar un mapa específico
  • !status - Ver el estado actual de las solicitudes
